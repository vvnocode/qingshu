# OpenClaw 入门指南：从零开始

这篇文档专门写给**完全没接触过 OpenClaw** 的同事。不管你是行政、运营、HR 还是财务，不需要任何编程基础，跟着做就行。

---

## 1. 什么是 AI Agent？

你可能用过 ChatGPT、文心一言、通义千问这类 AI——给它发条消息，它回一段文字。**这就是传统 Chatbot**，它只能"聊天"。

**AI Agent 不一样。**

打个比方：

| | 传统 AI Chatbot | AI Agent (OpenClaw) |
|---|---|---|
| 像什么 | 像一个**客服**——你问什么它答什么，挂了电话它就忘了你 | 像一个**住在你电脑里的助手**——能帮你干活、记住你的习惯、到点自动办事 |
| 能做什么 | 回答问题、写文案 | 回答问题 + 查资料 + 操作文件 + 发消息 + 定时执行任务 + …… |
| 记性 | 聊完就忘 | 能记住你说过的话和偏好 |
| 主动性 | 你不问它不动 | 可以定时主动干活（比如每天早上给你发天气） |

OpenClaw 的核心能力：对话、工具调用、持久记忆、定时自动执行。

---

## 2. OpenClaw 能帮你做什么？

别被"技术"两个字吓到，来看看各岗位的真实场景：

### 行政

- 每天早上自动推送**天气预报 + 今日日程**到钉钉群
- 帮你整理会议纪要，提取待办事项
- 自动提醒同事提交周报

### 运营

- 定时抓取竞品网站数据，生成对比分析报告
- 每周自动汇总运营数据，生成可视化图表
- 监控社交媒体关键词，有异常及时通知

### HR

- 批量筛选简历，按条件打分排序
- 自动统计考勤数据，标记异常
- 新员工入职材料自动整理归档

### 财务

- 发票照片自动识别，录入到表格
- 报销单自动审核格式和金额
- 定时生成费用汇总报表

### 通用场景

- **信息搜集整理**：帮你查资料、整理成文档
- **文档生成**：根据简单描述生成报告、方案、邮件
- **定时提醒**：任何你想定期做的事情
- **文件处理**：批量重命名、格式转换、数据清洗

## 3. 五分钟快速体验

### 第一步：安装 OpenClaw

打开你的终端（Mac 用"终端" app，Windows 用 PowerShell），复制粘贴下面的命令：

**方式一：一键安装脚本（推荐）**

```bash
curl -fsSL https://openclaw.ai/install.sh | bash
```

**方式二：通过 npm 安装**

```bash
npm i -g openclaw
```

> 💡 **提示**：不确定该用哪个？选方式一就对了，它会自动帮你搞定所有依赖。

### 第二步：首次配置

安装完成后，运行配置向导：

```bash
openclaw onboard
```

向导会引导你完成三件事：

1. **选择 AI 模型** —— 推荐先选免费的或公司统一提供的模型
2. **选择沟通渠道** —— 可以先用终端聊天，后面再接钉钉/飞书
3. **完成认证** —— 按提示填入 API Key 即可

### 第三步：第一次对话

配置好后，直接在终端输入：

```bash
openclaw tui
```

然后你就可以和 OpenClaw 聊天了！试试：

```
你好！请介绍一下你能帮我做什么？
```

它会告诉你它的能力范围。再试试：

```
帮我查一下今天北京的天气
```

注意：它不是只"回答"天气，而是调用了 `web_search` 工具去查询实时数据，然后返回结果。

### 第四步：创建第一个定时任务

这是 OpenClaw 区别于普通 Chatbot 的核心能力——**定时自动执行任务**。

在 OpenClaw 对话中输入：

```
帮我设置一个定时任务：每天早上8点查询并推送北京天气预报
```

OpenClaw 支持通过自然语言对话来创建定时任务（Cron Job），无需记忆命令行语法。它会理解你的意图，自动帮你配置好定时规则。明天早上 8 点，OpenClaw 就会自动查好天气推送给你。

## 4. "Hello World" 实操示例

### 示例 1：让 OpenClaw 帮你搜集整理信息

**场景**：老板让你调研一下市面上主流的项目管理工具。

在 OpenClaw 对话中输入：

```
帮我调研目前市面上主流的项目管理工具，包括飞书项目、Jira、Asana、Teambition、
Tower，从功能、价格、适用场景三个维度对比，输出一份 Markdown 表格。
```

OpenClaw 会：
1. 搜集各工具的信息
2. 整理对比维度
3. 输出一份格式清晰的表格

你可以直接复制粘贴到文档里。

### 示例 2：让 OpenClaw 帮你生成一份工作周报

**场景**：又到周五了，该写周报了。

```
帮我写一份本周工作周报。
本周完成的事情：
1. 完成了Q2市场推广方案
2. 跟进了3个客户的合同签署
3. 组织了一次部门team building
下周计划：
1. 启动Q2推广执行
2. 新客户拜访
```

或者

```
基于本周我们的聊天记录，帮我写一份工作汇报。
格式如：xxxxx。
我的补充内容：xxxxxx。
```

OpenClaw 会帮你润色成一份结构清晰、措辞专业的周报，你只需要检查一下就能发出去。

### 示例 3：设置每天早上的 Morning Brief

**场景**：希望每天早上上班前就收到一份信息简报。

```bash
每天早上8:30，帮我做以下事情：
1. 查询今天北京的天气和穿衣建议
2. 总结今天的科技新闻头条（3-5条）
3. 提醒我今天的日程安排
然后把结果整理成一份简报推送给我
```

从此，你每天打开钉钉/终端就能看到一份专属的 Morning Brief！

## 5. 常见问题 FAQ

### 它安全吗？

**安全。** OpenClaw 默认在你的电脑（或公司服务器）本地运行，你的数据不会上传到外部服务器。它不是"云服务"，而是跑在你自己机器上的程序。

### 要花多少钱？

- **可以零成本体验**：搭配免费模型（如部分开源模型）即可使用
- **公司统一采购**：如果公司已配置 API Key，直接使用不产生额外费用
- **个人购买**：各模型按 Token 计费，日常使用通常每月几十元
- 详见 → [06-大模型配置与费用优化](./06-OpenClaw%20大模型配置与费用优化.md)

### 💻 需要写代码吗？

**不需要！** 你只要用正常的中文和它对话就行。它能理解自然语言。

当然，如果你会写代码，能玩出更多花样。

### 手机上能用吗？

**可以！** OpenClaw 支持接入钉钉、飞书、微信、Slack 等 IM 通道。配置好后，你直接在手机上的钉钉/飞书里和它聊天就行。

详见 → [04-通道配置（钉钉）](./04-OpenClaw%20通道配置（钉钉）.md)

### 😰 它会不会删我的文件？

OpenClaw 确实有操作文件的能力，但有多层保护：

1. **权限控制**：可以配置它只能读取、不能删除
2. **操作确认**：关键操作会先询问你
3. **审计日志**：所有操作都有记录可查
4. **沙箱模式**：可以限制它只能访问指定目录

建议参考 → [10-规范与安全准则](./10-OpenClaw%20规范与安全准则.md) 进行配置。

### 🤔 它和 ChatGPT 有什么区别？

ChatGPT 只能在网页里聊天。OpenClaw 跑在本地，能直接操作文件、执行命令、定时干活、记住你的偏好。ChatGPT 定位是聊天工具，OpenClaw 定位是可执行任务的 AI 助手。

### 🤔 它和 Dify / Coze 有什么区别？

| 维度 | OpenClaw | Dify / Coze |
|------|----------|-------------|
| **部署方式** | 本地优先，数据不出境 | 云端 SaaS（Dify 可自部署） |
| **核心定位** | Agentic AI——自主执行、定时干活、多 Agent 协作 | 工作流编排、对话机器人搭建 |
| **扩展方式** | Skill（Markdown）+ Plugin（代码）+ MCP 工具 | 可视化拖拽工作流 + API 节点 |
| **数据隐私** | 完全本地，零云端依赖 | 依赖云端服务（Dify 自部署除外） |
| **适用场景** | 需要 Agent 自主执行系统操作、定时任务、多 Agent 协作 | 快速搭建对话机器人、简单工作流 |

Dify/Coze 侧重可视化 AI 应用搭建，OpenClaw 侧重本地部署的 Agentic 任务执行。

### 它回复很慢怎么办？

- 检查网络连接（如果用的是云端模型）
- 尝试换一个更快的模型（如 DeepSeek-V3 或 MiniMax-M2.5）
- 减少单次任务的复杂度，拆成多步完成
